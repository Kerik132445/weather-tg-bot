# 🌤️ Weather Telegram Bot & FastAPI Backend

Асинхронный Telegram-бот для отслеживания погоды с возможностью добавления городов в избранное. Проект реализован на раздельной архитектуре: бот выступает в роли клиента, а вся бизнес-логика и работа с данными вынесены на FastAPI бэкенд.

## 🚀 Технологический стек

* **Backend:** FastAPI, Uvicorn, HTTPX (для асинхронных запросов к WeatherAPI)
* **Telegram Bot:** Aiogram 3.x (FSM, CallbackData, Inline Keyboards)
* **Database:** SQLite / Асинхронный драйвер (через database.py)

## 📌 Ключевые возможности

* 🔍 Поиск текущей погоды в любом городе мира на русском языке через стороннее API.
* ⭐️ Динамическое добавление городов в список избранного.
* 🌡️ Автоматическое форматирование вывода: интеграция тематических эмоджи в зависимости от температуры и погодных условий.
* 🗑️ Интерактивное управление избранным: удаление карточки города прямо из чата с автоматическим стиранием сообщения.

# 2. Настройка бэкенда (FastAPI)
  Перейдите в директорию с бэкендом и запустите сервер:
  
    Bash
    uvicorn main:app --reload
    Бэкенд будет доступен по адресу: http://127.0.0.1:8000

# 3. Запуск бота (Aiogram)
  Убедитесь, что в коде указан ваш корректный TOKEN, и запустите скрипт:

    Bash
    python bot.py
   
# 📐 Архитектура проекта
Бот отправляет запросы на локальный сервер FastAPI.
Бэкенд выполняет асинхронные запросы к WeatherAPI, обрабатывает данные, кэширует/сохраняет города в базу данных SQLite и возвращает боту готовый структурированный JSON.
